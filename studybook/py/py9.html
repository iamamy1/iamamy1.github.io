# 集合

# 定义

set 关键字 无序的不重复的元素

作用：不重复特点

声明集合：set

s1 = set()  # 创建空集合只能用这种方式
s2 = {1, 3, 7}  # 创建非空集合可以用这种方式
# 字典：{key:value, key:value,......}
# 集合：{元素1, 元素2, 元素3,.....}

# 增删改查

  1. 增加

    add() 添加一个元素

    update() 可以添加多个元素，可以通过列表或者元组添加

    s1 = set()
s1.add('hello')
s1.add('小猪佩奇')
s1.add('iamamy1')
print(s1)  # {'hello', 'iamamy1', '小猪佩奇'}
t1 = ('aa', 'bb')
s1.update(t1)
print(s1)  # {'小猪佩奇', 'aa', 'iamamy1', 'bb', 'hello'}
t2 = ['aaa', 'bbb']
s1.update(t2)
print(s1)  # {'小猪佩奇', 'aa', 'bbb', 'iamamy1', 'bb', 'hello', 'aaa'}

  2. 删除

    remove() 如果元素存在则删除，不存在则报keyError

    pop() 随机删除（一般删除第一个元素）

    clear() 清空

    discard() 类似remove() 但是在移除不存在的元素的时候不会报错

    s1 = {'小猪佩奇', 'aa', 'bbb', 'iamamy1', 'bb', 'hello', 'aaa'}
s1.remove('iamamy1')
s1.discard('iamamy')  # 删除不存在的元素也不会报错
print(s1)  # {'bbb', 'aaa', 'hello', '小猪佩奇', 'bb', 'aa'}
s1.pop()
print(s1)  # {'hello', 'aaa', 'aa', 'bbb', 'bb'}
s1.clear()
print(s1)  # set()

# 可变和不可变

不可变： 对象所指向的内存中的值是不可以改变的

不可变类型：int str float 元组tuple

可变： 对象所指向的内存中的值是可以改变的

可变类型：字典dict 列表list
