# 元组

类似列表（当成容器来使用）

特点：

  1. 定义的符号：()，元素之间用逗号隔开
  2. 元组中的内容不可修改
  3. 关键字：tuple

tuple() ：元组类型的强制类型转换

查询： 下标 index

切片： [ : ]

最大值： max( )

最小值： min( )

求和： sum( )

求长度： len( )

排序： sorted( ) 可以排序，但是返回结果是列表类型的

符号： + * is not is in not in

# 元组中的函数

t.count( e ) ：找个数，返回元组 t 中元素 e 的个数。

t.index( e ) ：找位置，返回元组 t 中元素 e 的下标，如果元组 t 中没有元素 e ，则报错。

# 拆包和装包

赋值：装包

*y = 4,8,6

系统：散列 ---> [ ] ---> [4,8,6]

打印：拆包

print(*[4,8,6]) ---> 拆包 ---> 4,8,6

# 拆包和装包
t1 = (4, 7, 3)
# a,b = t1  # ValueError: too many values to unpack (expected 2)
# x, y, z = (6,)  # alueError: not enough values to unpack (expected 3, got 1)
a, b, c = t1
print(a, b, c)  # 4 7 3

# 变量个数与元组个数不一致
t1 = (2, 5, 8, 9, 7)
a, *_, c = t1
print(a, c, _)  # 2 7 [5, 8, 9]
a, b, *c = t1
print(a, b, c)  # 2 5 [8, 9, 7]  *c表示未知个数0~n
print(*c)  # 8 9 7   拆

t1 = (9,)
a, *b = t1
print(a, b)  # 9 []
