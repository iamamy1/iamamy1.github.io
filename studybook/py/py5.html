[ ] 看可以结合位置，获取字母 特点：只能获取一个字母

位置都是从 0 开始，位置也会称为下标或者索引

[ start : end ] 截取字符串，包前不包后

省略用法 [ start : ] 只要省略的是后面，表示一直取到字符串的末尾

省略用法 [ : end ]只要省略的是前面，表示从0开始取值

负数：从后往前 从 -1 开始

[ start : end : step ] step表示方向和步长

filename = 'picture.png'
print(filename[1])  	# i
print(filename[3:7])    # ture
print(filename[3:])     # ture.png
print(filename[:7])     # picture
print(filename[8:-1])   # pn
print(filename[:-2])    # picture.p
print(filename[-1:])    # g
print(filename[::])     # picture.png
print(filename[::-1])   # gnp.erutcip
print(filename[5:0:-1]) # rutci
print(filename[0:5:1])  # pictu
print(filename[::2])    # pcuepg
print(filename[::-3])   # g.ui

# 大小写

capitalize()：将字符串的第一个字符转换为大写

title()：返回“标题化”的字符串，就是说所有单词都是以大写开始，其余字母均为小写

istitle()：如果字符串中所有的单词拼写首字母是否为大写，且其他字母为小写则返回 True，否则返回 False

upper()：将字符串中的小写字母转为大写字母，返回将字符串中所有小写字符转换为大写后生成的字符串

isupper()：检测字符串中所有的字母是否都为大写

lower()：将字符串中的大写字母转为小写字母，返回将字符串中所有大写字符转换为小写后生成的字符串

islower()：检测字符串中所有的字母是否都为小写

# 查找与替换

find( str, beg, end )：str 要查找的字符子串 beg 查找起始位置 end 查找结束位置

检测字符串中是否包含子字符串 str ，如果指定 beg（开始） 和 end（结束） 范围，则检查是否包含在指定范围内，如果指定范围内如果包含指定索引值，返回的是索引值在字符串中的起始位置。如果不包含索引值，返回-1

rfind( str, beg, end )：str 要查找的字符子串 beg 查找起始位置 end 查找结束位置

返回字符串最后一次出现的位置，如果没有匹配项则返回-1 从右侧检索

index( str, beg, end )：str 要查找的字符子串 beg 查找起始位置 end 查找结束位置

检测字符串中是否包含子字符串 str ，如果指定 beg（开始） 和 end（结束） 范围，则检查是否包含在指定范围内，该方法与 python find()方法一样，只不过如果str不在 string中会报一个异常

rindex(str, beg, end )：str 要查找的字符子串 beg 查找起始位置 end 查找结束位置

返回子字符串 str 在字符串中最后出现的位置，如果没有匹配的字符串会报异常，可以指定可选参数[beg:end]设置查找的区间

replace( old , new, [, max] )：

把字符串中的 old（旧字符串） 替换成 new(新字符串)，如果指定第三个参数max，则替换不超过 max 次

str.count(sub, start= 0,end=len(string))：用于统计字符串里某个字符或子字符串出现的次数。可选参数为在字符串搜索的开始与结束位置

  • sub -- 搜索的子字符串。
  • start -- 字符串开始搜索的位置。默认为第一个字符,第一个字符索引值为0。
  • end -- 字符串中结束搜索的位置。字符中第一个字符的索引为 0。默认为字符串的最后一个位置。

# 编码

主要用于网络应用：

encode( encoding='UTF-8',errors='strict' ) 编码

以指定的编码格式编码字符串。errors参数可以指定不同的错误处理方案。

  • encoding -- 要使用的编码，如: UTF-8。
  • errors -- 设置不同错误的处理方案。默认为 'strict',意为编码错误引起一个UnicodeError。 其他可能得值有 'ignore', 'replace', 'xmlcharrefreplace', 'backslashreplace' 以及通过 codecs.register_error() 注册的任何值

decode( encoding="utf-8", errors="strict" ) 解码

以指定的编码格式解码 bytes 对象。默认编码为 'utf-8'。

  • encoding -- 要使用的编码，如"UTF-8"。
  • errors -- 设置不同错误的处理方案。默认为 'strict',意为编码错误引起一个UnicodeError。 其他可能得值有 'ignore', 'replace', 'xmlcharrefreplace', 'backslashreplace' 以及通过 codecs.register_error() 注册的任何值。

# 合并拆分

str.join( sequence )：用于将序列中的元素以指定的字符连接生成一个新的字符串

str 指定字符 sequence 序列

s1 = "-"  # 分隔字符
seq = ("r", "u", "n", "o", "o", "b")  # 字符串序列

print(s1.join(seq))  # r-u-n-o-o-b

print('_'.join('abc'))  # a_b_c

list1 = ['a', 'b', 'c', 'd']
print(s1.join(list1))  # a-b-c-d

strip( [chars] ) ：删除字符串头尾指定的字符（默认为空格）或字符序列

  • chars -- 指定截取的字符。

lstrip( [chars] ) ：删除 string 字符串左侧的指定字符（默认为空格）

  • chars -- 指定截取的字符。

rstrip( [chars] )：删除 string 字符串右侧的指定字符（默认为空格）

  • chars -- 指定截取的字符。

split( str="", num=string.count(str) )：通过指定分隔符对字符串进行切片，如果第二个参数 num 有指定值，则分割为 num+1 个子字符串，将分割后的字符串保存到列表中，返回的是一个列表

  • str -- 分隔符，默认为所有的空字符，包括空格、换行(\n)、制表符(\t)等。
  • num -- 分割次数。默认为 -1, 即分隔所有。
