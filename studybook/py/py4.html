# 4.2 循环语句（for和while）

应用场景：

  1. 猜大小：反复的猜

  2. 消消乐：反复充值

  3. 用户登录：登录多次

# for 语法：

  1. 循环结构：

    for 变量名 in 集合 :

    ​ 语句

**补充：**使用系统给定range( )完成范围指定：

print( range( 8 ) ) # range ( 0, 8 ) 包含0，但是不包含8

range( ) 有以下三种用法：

  • range( n ) ~ range( 0, n ) : [ 0 , n )
  • range( m, n ) ~ range( start, end ) : [ m , n )
  • range( m, n ，step) ~ range( start, end, step ) : [ m , n ) step：步长

  1. for . . . else

    else : 适用于 for 执行完或者没有循环数据时，需要做的事情

    结构：

    for i in 范围：

    ​ 有数据执行的语句

    else:

    ​ 没有数据执行的语句

**补充：**对于以下语句

for i in range( 3 ) : ​ if 条件 : ​ 语句 ​ else : ​ 语句 else : ​ print('-----------------')

range 的范围正常执行完毕，而没有异常 break 跳出，就可以执行 else , 只要有 break 执行了就不会执行 else

# 4.3 跳转语句

pass：空语句 只有缩进而缩进的内容还不确定的时候，此时为了保证语法的正确性，就可以使用pass占位

break：跳出循环结构

continue：跳过循环体中下方的语句不执行，直接进行下一次循环
