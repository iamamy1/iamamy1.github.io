**>=：**大于等于

**<=：**小于等于

**>：**大于

**<：**小于

**is：**用户对象的比较 即 id( ) 比较的是地址

is not：

***小问题：*为什么以上代码在源文件中和在交互式环境中运行结果不一样？

1.在源文件中：变量池 复用 地址相同

2.在交互式环境中：小整数[-5, 256]可以复用（小整数对象池）；大整数不行

小整数对象池：[-5, 256]

*大整数对象池：*终端（交互式环境）是每次执行一次，所以每次得大整数都重新创建，而在pycharm中，每次运行是所有代码都加载到内存中，属于一个整体，所以这时候会有一个大整数对象池，即处于一个代码块的大整数是同一个对象。

# 3.6 三元运算符(三目运算符)

C语言格式：表达式 ? 真 : 假

python格式：结果 if 表达式 else 结果

判断表达式是 True 还是 False

如果是 True 则 将 if 前面的内容进行运算

如果是 False 则 将 if 后面的内容进行运算
