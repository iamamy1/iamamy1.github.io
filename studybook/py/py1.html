# 变量和数据类型

# 1.1 变量命名格式

变量名 = 值

# 1.2变量名的命名规范

字母，数字，下划线，且数字不能开头

严格区分大小写

不能使用关键字

见名知义：驼峰命名法（小驼峰，大驼峰），下划线命名法

# 1.3 变量的数据类型

数字：int , long , float , complex

布尔类型（boolean）：True , False

字符串：String

列表：List

元组：Tuple

字典：Dictionary

# 1.4 判断变量的数据类型

type('变量名')

# 1.5 输出语句

print( value,value , , sep = ' ', end = '\n')

sep默认的分割是空格

end默认的结尾字符是换行

# 1.6 输入语句

input( ' 输入提示 ' ) # 阻塞式

**注意：**键盘输入的都是字符串类型，可以通过 a = input( ' 输入提示 ' ) 将输入内容赋值给变量a

# 1.7 类型转换

以变量名：a 为例

str ----> int ： int(a) 但是如果是 ' 9.9 ' 而且是字符串类型转换成int的时候报错了

str ----> float：float(a)

int ----> str： str(a)

float ----> str：str(a)

int ----> float：float(a)

float ----> int：int(a) 只不过float类型中小数点后面的数字被抹掉了

bool ----> int：True ----> 1 False ----> 0

bool ----> float：True ----> 1.0 False ----> 0.0

bool ----> str：True ----> 'True' False ----> 'False'

# 1.8 转义字符

\n 换行

\t 制表符

\'

\"

\r 回车

r'字符串' 原样输出字符串的内容，即使有转义字符也不会转义
