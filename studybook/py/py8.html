# 字典

特点：

  1. 符号：{ }
  2. 关键字：dict
  3. 保存的元素是 键：值对 key：value （key在字典中是唯一的）
  4. 是一种映射类型定义

# 定义

dict1 = {}  # 空字典
dict2 = dict()  # 空字典
dict3 = {'name': 'lucky', 'age': 18}
print(dict3)
dict4 = dict([('name', 'lucky'), ('age', 18)])
print(dict4)
dict5 = dict([(1, 2), (3, 4), (5, 6), (7, 8)])
print(dict5)

# 增加/修改元素

格式：dict[key] = value

按照上面的格式，如果在字典中存在同名的key，则发生覆盖（后面的值覆盖原来的）。

如果没有同名的key，则实现添加功能（key:value添加到字典中）

# 查找元素

dict[key] 字典中找元素根据key ， 如果key在字典中不存在则报出KeyError

dict.get(key) ： 返回指定key键的值，如果key在字典中不存在，返回None

dict.get(key, default) ： 返回指定key键的值，如果key在字典中不存在，返回默认值default

dict.items() ：以列表中包含元组的方式返回字典中内容 [ (key:value), (key:value), (key:value) ]

dict.values() ：取出字典中所有值，保存到列表中

dict.keys() ：取出字典中所有key键，保存到列表中

对字典进行遍历：

dict3 = {'name': 'lucky', 'age': 18}
for i in dict3:
    print(i)  # name  age
# 单独遍历字典的结果是：字典的key

# 删除元素

del dict[ key ] 删除字典中键为key的键值对， 如果key在字典中不存在则报出KeyError

dict.pop(key[,default]) ：根据key删除字典中的键值对，只要删除成功，返回值是删除成功的键值对的值value。

​ pop的默认值，往往是在删除的时候没有找到对应的key，则返回default默认值。

dict.popitem() ：随机返回并删除字典中的最后一对键和值。如果字典已经为空，却调用了此方法，就报出KeyError异常。返回的是删除的键值对。

dict.clear() ：用于删除字典内所有元素，用法同列表的clear

# 字典的内置函数

dict.items() ：以列表中包含元组的方式返回字典中内容 [ (key:value), (key:value), (key:value) ]

dict2 = {'张三': 100, '李四': 100, '王五': 89, '赵六': 99}
print(dict2.items())  # dict_items([('张三', 100), ('李四', 100), ('王五', 89), ('赵六', 99)])
for i in dict2.items():
    print(i)  # ('张三', 100)
for key, value in dict2.items():
    if value > 90:
        print(key)  # 张三  李四  赵六

dict.get(key) 返回指定key键的值，如果key在字典中不存在，返回None

dict.get(key, default) 返回指定key键的值，如果key在字典中不存在，返回默认值default

dict.pop(key[,default]) ：删除字典给定键 key 所对应的值，返回值为被删除的值。key值必须给出。 否则，返回default值。

dict.popitem() ：随机返回并删除字典中的最后一对键和值。如果字典已经为空，却调用了此方法，就报出KeyError异常。返回的是删除的键值对。

dict.clear() ：用于删除字典内所有元素，用法同列表的clear

其他的内置函数：

dict1.update(dict2) : 用dict2中的内容更新dict1

dict1 = {0: 'tom', 1: 'jack', 2: 'lucy'}
dict2 = {0: 'lily', 4: 'ruby'}
dict1.update(dict2)
print(dict1)  # {0: 'lily', 1: 'jack', 2: 'lucy', 4: 'ruby'}

dict.fromkeys(seq[, value]) ：用于创建一个新字典，以序列 seq 中元素做字典的键，value 为字典所有键对应的初始值，如果没有指定默认的value则用None

list1 = [1, 2, 3, 4, 5]
new_dict1 = dict.fromkeys(list1)
print(new_dict1)  # {1: None, 2: None, 3: None, 4: None, 5: None}
new_dict2 = dict.fromkeys(list1, False)
print(new_dict2)  # {1: False, 2: False, 3: False, 4: False, 5: False}
