# 6.2 Basics of deep learning

Supervised learning: we are given input x, and the goal is to predict label y

Formulate the task as an optimization problem：

  • 我们定义一个拟合函数(objective function)
  • 然后根据损失函数求出损失，以此来衡量拟合函数的好坏。

# Loss function

损失函数有很多种，比如L2 loss，交叉熵等等

One common loss for classification: cross entropy (CE) $$ CE(y,f(x))=-∑_{i=1}^C(y_ilogf(x)_i) $$ optimize the objective function : gradient vector(梯度是沿最大增长方向的方向导数) $$ ∇_Θℒ=(\frac{∂ℒ}{∂Θ_1},\frac{∂ℒ}{∂Θ_2},...) $$

# Gradient Descent

可以使用梯度下降对拟合函数进行多次迭代优化

  • Iterative algorithm: repeatedly update weights in the (opposite) direction of gradients until convergence（梯度一点一点缓慢下降，直到收敛，梯度为0时取得极值）
  • Ideal termination condition: 0 gradient. In practice, we stop training if it no longer improves performance on validation set

Stochastic gradient descent (SGD)

At every step, pick a different minibatch Β containing a subset of the dataset, use it as input x

minibatch SGD：

  • Batch size: the number of data points in a minibatch
  • Iteration: 1 step of SGD on a minibatch
  • Epoch: one full pass over the dataset(iterations is equal to ratio of dataset size and batch size)

两种不同的梯度下降方法：

SGD：匀速，1阶导，表示速度，得到局部最优结果

Adam：2阶导，表示加速度，期望得到全局最优，只是有可能得到全局最优结，可能会导致不收敛，而SGD一定收敛，可以先用Adam，将要收敛时再用SGD

# Back-propagation

Forward propagation:compute ℒ given x

Back-propagation: obtain gradient ∇Θℒ using a chain rule

Use of chain rule to propagate gradients of intermediate steps, and finally obtain gradient

chain rule:(链式求导法对于矩阵求导仍旧适用) $$ \frac{dz}{dx}=\frac{dz}{dx}·\frac{dy}{dx} $$

# non-linearity

  • Rectified linear unit (ReLU) ReLU(x) = max(x, 0)
  • Sigmoid

$$ σ(x)=\frac{1}{1+e^{-x}} $$

Multi-layer Perceptron(MLP)

Each layer of MLP combines linear transformation and non-linearity: $$ x^{l+1}=σ(W_lW_lx^l+b^l) $$

  • where Wl is weight matrix that transforms hidden representation at layer l to layer l + 1
  • bl is bias at layer l, and is added to the linear transformation of x
  • σ is non-linearity function

# A Naive Approach

  • Join adjacency matrix and features
  • Feed them into a deep neural net

issue：

  1. O(|V|) parameters(参数量大)

  2. Not applicable to graphs of different sizes

  3. Sensitive to node ordering

# Convolutional Networks

Node’s neighborhood defines a computation graph

# Aggregate Neighborhood

Generate node embeddings based on local network neighborhoods

  • Network neighborhood defines a computation graph
  • Every node defines a computation graph based on its neighborhood

Basic approach: Average neighbor messages and apply a neural network

  • ℎv(l)：the hidden representation of node v at layer l
  • Wk: weight matrix for neighborhood aggregation
  • Bk: weight matrix for transforming hidden vector of self（一个偏置项）

# Model Design

  • Define a neighborhood aggregation function（聚合函数）
  • Define a loss function on the embeddings（损失函数）
  • Train on a set of nodes, i.e., a batch of compute graphs（训练）
  • Generate embeddings for nodes as needed（生成嵌入）
